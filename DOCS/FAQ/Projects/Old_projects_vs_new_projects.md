# Old projects vs new projects

This article aims to compare our old project module (which we're gradually deprecating) with our brand new, frictionless projects module. Here you'll find an overview of all features and if they are already present in our new module or not. 

 

As a user of the old project module, this overview can help you decide if you want to start migrating to the new module already, [which you'll find more information about in this article.](https://support.focus.teamleader.eu/hc/en-150/articles/25696033231377-Migrate-to-our-new-project-module)

 

Do you prefer to watch our instructive webinar? You can do so here. Find the slides of the presentation [as an attachment to this article as well.](https://support.focus.teamleader.eu/hc/en-150/article_attachments/25696135318929)

 

![Tutorial title slide: Get work done on a solid foundation - Projects in Teamleader Focus.](https://support.focus.teamleader.eu/hc/article_attachments/25696120131857)

 

    The ❌symbol means a certain feature isn't available (yet) in the new projects module. In the upcoming months, more and more features will be developed and added to the new module.
