# How To: Using the LinkedIn plug-in in Teamleader Focus

![](https://support.focus.teamleader.eu/hc/article_attachments/25690800821009)

 

In this article we will:

- Install the plug-in

- Add contacts

## Install the plug-in

- Go to Teamleader Focus. Navigate to your **user icon** in the top right-hand corner and click on “**Integrations/Marketplace**”.

- Search for “LinkedIn” in the search field

- Choose “**LinkedIn**” and click “**Add**”

- The Chrome web store pops up. Click “**Add to Chrome**” and “**Add Extension**”

 

## Add contacts

- Go to the profile page of the person you want to add to Teamleader Focus

- Click on the “**Add to Teamleader**” button above the profile picture 
![](https://support.focus.teamleader.eu/hc/article_attachments/25690796326929)

- The person is now added as a contact to Teamleader Focus. The information added is the following: 

**First name** of the contact

- 
**Last ****name **of the contact

- If you want to, you can also add the **company/companies **they work for to Teamleader Focus

In this case, the **function(s) **of the contact is also added

           

Now that your LinkedIn connection is added in Teamleader Focus you’ll be able to:

- Recognise Teamleader Focus contacts in LinkedIn

- See a LinkedIn logo next to a contact’s name in Teamleader Focus. Clicking on this will take you directly to the contact’s profile page
