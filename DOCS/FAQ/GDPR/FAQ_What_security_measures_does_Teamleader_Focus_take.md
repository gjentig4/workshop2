# FAQ: What security measures does Teamleader Focus take?

![](https://support.focus.teamleader.eu/hc/article_attachments/25692781659025)

You can read about all our security measures in the DPA. Some examples:

## Access Control of Processing Areas (Physical)

Web applications, communications and database servers of Teamleader Focus are located in secure data centers in Ireland, which are operated by Amazon Web Services, Inc. with whom Teamleader Focus has signed the AWS data processing addendum in order to be compliant with the standards and obligations as set forth in the Privacy Legislation.

## Access Control to Personal Data Processing Systems (Logical)

Teamleader has implemented suitable measures to prevent its Personal Data Processing systems from being used by unauthorized persons.

## Transmission Control

Teamleader has implemented suitable measures to prevent Personal Data from being read, copied,altered or deleted by unauthorized parties during the transmission thereof or during the transport of the data media.

More information on our [security measures can also be found on our website](https://www.teamleader.eu/legal/security-teamleader-focus).
