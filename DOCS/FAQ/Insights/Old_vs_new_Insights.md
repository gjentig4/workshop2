# Old vs new Insights

Are you looking for a specific report from the old Insights module? Here we list how to recreate or find specific reports from the old insights module in the[ new Insights module](https://support.focus.teamleader.eu/hc/en-150/articles/28006164415889-How-To-Insights-in-Teamleader-Focus).

# Deals
 

**Old**
**New**

**Deals report**

- Deals created count

- Deal amount during a specific period per status

Deals created and deal amounts in a specific period:

- 
Start from [Deals across Phases](https://focus.teamleader.eu/insights/library-reports/3958953f-63ad-4585-95bd-4c911df622e6)

- Change grouping to Date Added

- Add deal count and deal amounts in header

- Filter by deal status or deal phase

**Funnel Report**: deals moving through phases over a specific period of time

Start from [Deals across Phases](https://focus.teamleader.eu/insights/library-reports/3958953f-63ad-4585-95bd-4c911df622e6)

- Filter on date added between period

- Add deal count and deal amounts in header
