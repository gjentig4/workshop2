# Getting started: What is a deal?

![A 'LET'S START' introductory banner with a paper plane illustration on a light blue background.](https://support.focus.teamleader.eu/hc/article_attachments/25691002060177)

 

Following up on potential customers? You can use Teamleader Focus' deal module for that. To keep things nice and simple: a deal basically shows the steps that need to be taken to convert an incoming lead into a customer. [Creating](https://support.focus.teamleader.eu/hc/en-150/articles/25693612763409-FAQ-How-can-I-create-a-quotation-in-Teamleader-Focus) and sending a quotation for example, is one of the steps that you'll need to take to convert a lead into a paying customer.
