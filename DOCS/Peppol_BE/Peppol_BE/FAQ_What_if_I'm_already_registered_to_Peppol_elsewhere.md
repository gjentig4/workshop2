# Only send documents

You can send e-invoices through Peppol via various software tools, but you can only receive e-invoices in one tool. If you're already registered somewhere else for receiving, we will detect this and only activate Peppol for sending documents. You can always activate receiving documents at a later point in time though by clicking on **Activate receiving documents **next to 'Receiving'.

Â 

- If you already registered for receiving Peppol documents in Teamleader Focus but want to deactivate receiving, then you first need to deactivate Peppol fully in Teamleader Focus, register for receiving in the other tool and then enable Peppol for sending only again in Teamleader Focus.

- If you're not registered anywhere else but still only want to use Teamleader Focus for sending e-invoices, you have to first register to receive via Peppol in another tool. Afterwards, you can still decide to also receive via Teamleader Focus.
